      PG Accommodation Finder Website
      Overview:
      The PG Accommodation Finder is a responsive web application designed to help users search for available paying guest accommodations based on their location, budget, and preferences. The platform connects landlords offering PG spaces with potential tenants, offering a seamless experience for both parties.
      Features:
      Search and Filters: Users can search for PGs based on location, price range, and amenities like Wi-Fi, meals, and laundry services. Responsive Design: The website adapts to all devices, ensuring a smooth user experience on desktops, tablets, and smartphones. User Authentication: Secure login and registration system for both tenants and landlords. Property Listings: Landlords can create, update, and delete PG listings, complete with photos, descriptions, and pricing. Reviews and Ratings: Tenants can leave reviews and ratings for PG accommodations to help others make informed decisions. Interactive Map Integration: Integration with Google Maps API to display PGs on a map for easy location-based searching.
      Technologies Used:
      Front-end: HTML5, CSS3, JavaScript, Bootstrap for responsive design Back-end: Node.js, Express.js for handling server-side logic Database: MongoDB for storing user data, PG listings, and reviews APIs: Google Maps API for map integration Authentication: JWT (JSON Web Tokens) for secure user authentication My Contribution: I was responsible for designing the front-end interface, ensuring the website was fully responsive and user-friendly. I also worked on integrating the Google Maps API for interactive location-based searches and collaborated on the back-end to implement user authentication and database management. Through this project, I honed my skills in full-stack web development, problem-solving, and collaboration in a team setting.
